SECTION 1: PROJECT INTRODUCTION QUESTIONS
Q1. Explain your project in one minute.

Answer

Sir, our project aims to improve the usability of cloud-contaminated LISS-IV satellite imagery.

Instead of treating cloud removal as a simple image enhancement problem, we treat it as a multi-modal Earth surface reconstruction problem.

The model takes:

Cloudy LISS-IV imagery
Temporal neighboring optical images
Sentinel-1 SAR imagery

and reconstructs the most probable cloud-free LISS-IV image using a conditional diffusion architecture.

The output includes:

Cloud-free FCC image
Pixel-level uncertainty map

which allows analysts to know both the reconstructed image and its confidence.

Q2. What problem are you solving?
Answer

The main challenge is that LISS-IV possesses high spatial resolution but only three spectral bands:

Green
Red
NIR

It lacks:

SWIR
Thermal
Water vapor channels

which are critical for cloud discrimination.

As a result:

Atmospheric correction becomes difficult
Cloud detection becomes difficult
Cloud removal becomes extremely difficult

Our system compensates for these missing dimensions using:

Temporal information
SAR information
Diffusion-based reconstruction
SECTION 2: LISS-IV QUESTIONS
Q3. Why specifically LISS-IV?
Answer

LISS-IV provides:

5.8 m spatial resolution
High-quality agricultural monitoring
Detailed land-cover mapping

However it suffers from:

Limited spectral richness
Atmospheric sensitivity

Thus it presents an excellent real-world challenge where modern AI can contribute.

Q4. Why can't existing atmospheric correction methods solve this?
Answer

Atmospheric correction removes atmospheric effects.

It cannot reconstruct information hidden beneath thick clouds.

For example:

Haze → correction possible
Thin cloud → partial recovery possible
Dense cloud → information absent

Our project targets the third category.

Q5. Explain FCC.
Answer

FCC means False Color Composite.

For LISS-IV:

Display Red   ← NIR
Display Green ← Red
Display Blue  ← Green

Result:

Vegetation → Bright Red
Water → Dark Black
Urban → Cyan
Bare Soil → Brown

This enhances vegetation analysis.

SECTION 3: AI QUESTIONS
Q6. Why has AI failed previously?
Answer

AI fails because cloud removal is fundamentally different from denoising.

Denoising:

Signal + Noise

Cloud obstruction:

Signal hidden entirely

Thus AI must infer missing information.

Without additional evidence:

Temporal imagery
SAR imagery
Physical constraints

the problem becomes ill-posed.

Q7. Why not use a CNN?
Answer

CNNs perform single-step reconstruction.

They:

struggle with large cloud regions
hallucinate structures
lack temporal memory

Diffusion models iteratively refine predictions and preserve detail better.

Q8. Why diffusion instead of GAN?
Answer

GANs often produce:

Visually realistic
but physically incorrect

outputs.

Diffusion models:

are more stable
preserve structure better
produce fewer artifacts
achieve superior reconstruction quality

in modern remote sensing studies.

Q9. What is conditional diffusion?
Answer

The diffusion process is guided by external information.

Instead of generating an image randomly:

P(Image)

we generate:

P(Image | Conditions)

Conditions include:

Cloudy LISS-IV
SAR
Temporal images

Q10. Why use SAR?
Answer

SAR operates in microwave wavelengths.

Microwaves penetrate:

Clouds
Rain
Haze

Therefore SAR provides information even when optical sensors fail.

Q11. What SAR data will you use?

Most likely:

Sentinel-1

Open access

Q12. Can SAR directly replace optical imagery?
Answer

No.

SAR measures:

Backscatter

while LISS-IV measures:

Reflectance

They represent different physical phenomena.

Therefore SAR acts as complementary information rather than replacement.
